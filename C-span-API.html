C++ span

template<
    class T, // 元素类型；必须是完整对象类型且非抽象类
    std::size_t Extent = std::dynamic_extent // 序列中的元素数，或若它为动态则为 std::dynamic_extent
> class span;

类模板 span 所描述的对象能指代对象的相接序列，序列的首元素在零位置。 span 能拥有静态长度，该情况下序列中的元素数已知并编码于类型中，或拥有动态长度。可以用于防止数组传参时类型退化。

成员类型

成员类型 定义
element_type T
value_type std::remove_cv_t<T>
size_type std::size_t
difference_type std::ptrdiff_t
reference T&
const_reference const T&
pointer T*
const_pointer const T*
iterator 实现定义的遗留随机访问迭代器 、常量表达式迭代器兼遗留连续迭代器 ，其 value_type 为 value_type
reverse_iterator std::reverse_iterator<iterator>
const_reverse_iterator std::reverse_iterator<const_iterator>

成员常量

static constexpr std::size_t extent = Extent;

构造

构造函数

  1. 创建一个空容器. 构造空 span 使其 data() == nullptr 而 size() == 0 。

    constexpr span() noexcept;

  2. 构造作为范围 [first, first + count) 上的视图的 span ；产生的 span 有 data() == std::to_address(first) 与 size() == count 。

    template< class It >
explicit(extent != std::dynamic_extent) constexpr span( It first, size_type count );

  3. 构造作为范围 [first, last) 上的视图的 span ；产生的 span 有 data() == std::to_address(first) 与 size() == last-first 。

    template< class It, class End >
explicit(extent != std::dynamic_extent) constexpr span( It first, End last );

  4. 构造作为数组 arr 上的视图的 span ；产生的 span 有 size() == N 与 data() == std::data(arr) 。

    template< std::size_t N >
constexpr span( element_type (&arr)[N] ) noexcept;

template< class U, std::size_t N >
constexpr span( std::array<U, N>& arr ) noexcept;

template< class U, std::size_t N >
constexpr span( const std::array<U, N>& arr ) noexcept;

  5. 构造作为范围 range 上的视图的 span ；产生的 span 有 size() == std::ranges::size(r) 与 data() == std::ranges::data(range)。

    template< class R >
explicit(extent != std::dynamic_extent) constexpr span( R&& range );

  6. 从另一 span source 的转换构造函数；产生的 span 有 size() == source.size() 与 data() == source.data().

    template< class U, std::size_t N >
explicit(extent != std::dynamic_extent && N == std::dynamic_extent)
constexpr span( const std::span<U, N>& s ) noexcept;

  7. 默认化的赋值构造函数复制大小与数据指针；产生的 span 有 size() == other.size() 与 data() == other.data() 。

    constexpr span( const span& other ) noexcept = default;

operator=

赋值 other 给 *this 。此默认化的赋值运算符进行数据指针和大小的浅复制，即调用此函数后， data() == other.data() 且 size() == other.size() 。

constexpr span& operator=( const span& other ) noexcept = default;

元素访问

operator[]

返回位于指定位置 pos 的元素的引用。通过此运算符访问不存在的元素是未定义行为。

constexpr reference operator[](size_type idx) const;

front

返回到容器首元素的引用。在空容器上对 front 的调用是未定义的。

constexpr reference front() const;

back

返回到容器中最后一个元素的引用。在空容器上调用 back 导致未定义行为。

constexpr reference back() const;

data

返回指向作为元素存储工作的底层数组的指针。对于非空容器，返回的指针与首元素地址比较相等。

constexpr pointer data() const noexcept;

迭代器

begin - end

返回指向容器开头和结尾的迭代器。

constexpr iterator begin() const noexcept;constexpr iterator end() const noexcept;

rbegin - rend

返回指向容器开头和结尾的反向迭代器。

constexpr reverse_iterator rbegin() const noexcept;constexpr reverse_iterator rend() const noexcept;

观察器

empty

检查 span 是否为空。空则为true

[[nodiscard]] constexpr bool empty() const noexcept;

size

返回 span 中的元素数。

constexpr size_type size() const noexcept;

size_bytes

返回以字节表示的序列大小.

constexpr size_type size_bytes() const noexcept;

子视图

first

获得由序列首 N 个元素组成的子段.作为*this 的首 Count 个元素上的视图的 span r ，满足 r.data() == this->data() && r.size() == Count 。

template< std::size_t Count >constexpr std::span<element_type, Count> first() const;

constexpr std::span<element_type, std::dynamic_extent> first( std::size_t Count ) const;

last

获得由序列末 N 个元素组成的子段.

template< std::size_t Count > constexpr std::span<element_type, Count> last() const;

constexpr std::span<element_type, std::dynamic_extent> last( std::size_t Count ) const;

subspan

获得子段.获得 span ，它是此 span 的始于偏移 Offset 的 Count 个元素上的视图。若 Count 为 std::dynamic_extent ，则子段中的元素数为 size() - offset （即它结束于 *this 的结尾）。

template< std::size_t Offset, std::size_t Count = std::dynamic_extent > constexpr std::span<element_type, E > subspan() const;// 返回的 span 的长度模板实参 E ：
//		若 Count 不是 std::dynamic_extent 则为 Count ；
//		否则，若 Extent 不是 std::dynamic_extent ，则为 Extent - Offset ；
//		否则为 std::dynamic_extent 。
constexpr std::span<element_type, std::dynamic_extent>
	subspan( std::size_t Offset,std::size_t Count = std::dynamic_extent ) const;

非成员函数

as_bytes

转换 span 为对其底层字节的视图.以 {reinterpret_cast<const std::byte*>(s.data()), s.size_bytes()}构造的 span 。

template< class T, std::size_t N>std::span<const std::byte, S> as_bytes(std::span<T, N> s) noexcept;

as_writable_bytes

转换 span 为对其底层字节的视图.以 {reinterpret_cast<std::byte*>(s.data()), s.size_bytes()} 构造的 span 。

template< class T, std::size_t N>std::span<std::byte, S> as_writable_bytes(std::span<T, N> s) noexcept;

非成员常量

dynamic_extent

size_t 类型常量，指明 span 拥有动态长度

inline constexpr std::size_t dynamic_extent = std::numeric_limits<std::size_t>::max();

辅助模板

std::ranges::enable_borrowed_range 的此特化使得 span 满足 borrowed_range 。

template<class T, std::size_t Extent>inline constexpr bool ranges::enable_borrowed_range<std::span<T, Extent>> = true;

std::ranges::enable_view 的此特化使零或动态长度的 span 满足 view 。非零静态长度的 span 非 default_initializable 从而不是视图。

template<class T, std::size_t Extent>inline constexpr bool ranges::enable_view<std::span<T, Extent>> =    Extent == 0 || Extent == dynamic_extent;
